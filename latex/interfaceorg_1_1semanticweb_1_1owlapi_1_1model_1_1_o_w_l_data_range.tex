\subsection{Detailed Description}
Author\-: Matthew Horridge\par
 The University Of Manchester\par
 Bio-\/\-Health Informatics Group Date\-: 24-\/\-Oct-\/2006 

Represents a \href{http://www.w3.org/TR/2009/REC-owl2-syntax-20091027/#Data_Ranges}{\tt Data\-Range} in the O\-W\-L 2 Specification. 

A high level interface which represents a data range. Example of data ranges are datatypes (e.\-g. int, float, double, string, ...), complements of data ranges (e.\-g. not(int)), data enumerations (data one\-Ofs), datatype restrictions (e.\-g. int $>$ 3).
