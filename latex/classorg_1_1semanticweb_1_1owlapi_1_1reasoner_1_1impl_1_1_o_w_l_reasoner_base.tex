Gets the time out (in milliseconds) for the most basic reasoning operations. That is the maximum time for a satisfiability test, subsumption test etc. The time out should be set at reasoner creation time. During satisfiability (subsumption) checking the reasoner will check to see if the time it has spent doing the single check is longer than the value returned by this method. If this is the case, the reasoner will throw a \hyperlink{classorg_1_1semanticweb_1_1owlapi_1_1reasoner_1_1_time_out_exception}{org.\-semanticweb.\-owlapi.\-reasoner.\-Time\-Out\-Exception} in the thread that is executing the reasoning process. 

Note that clients that want a higher level timeout, at the level of classification for example, should start their own timers and request that the reasoner interrupts the current process using the \hyperlink{interfaceorg_1_1semanticweb_1_1owlapi_1_1reasoner_1_1_o_w_l_reasoner_a3062d390ff3ba053d7abb2fcd169e357}{interrupt()} method.

\begin{DoxyReturn}{Returns}
The time out in milliseconds for basic reasoner operation. By default this is the value of \hyperlink{}{Long\#\-M\-A\-X\-\_\-\-V\-A\-L\-U\-E}. 
\end{DoxyReturn}
