\subsection{Detailed Description}
Author\-: Matthew Horridge\par
 The University Of Manchester\par
 Bio-\/\-Health Informatics Group\par
 Date\-: 08-\/\-Oct-\/2007\par
\par


A general purpose interface, implementations of which can be used to renderer O\-W\-L objects (e.\-g. class expressions, axioms etc.) for presentation in user interfaces, colsole writing etc. etc. Many ontology renderers may also choose to implement this interface.
