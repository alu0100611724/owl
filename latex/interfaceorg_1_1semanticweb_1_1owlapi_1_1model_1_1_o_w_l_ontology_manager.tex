An {\ttfamily \hyperlink{interfaceorg_1_1semanticweb_1_1owlapi_1_1model_1_1_o_w_l_ontology_manager}{O\-W\-L\-Ontology\-Manager}} manages a set of ontologies. It is the main point for creating, loading and accessing ontologies. 

An {\ttfamily \hyperlink{interfaceorg_1_1semanticweb_1_1owlapi_1_1model_1_1_o_w_l_ontology_manager}{O\-W\-L\-Ontology\-Manager}} also manages the mapping betweem an ontology and its ontology document.
