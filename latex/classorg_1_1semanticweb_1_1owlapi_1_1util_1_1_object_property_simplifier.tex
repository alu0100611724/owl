\subsection{Detailed Description}
Author\-: Matthew Horridge\par
 The University Of Manchester\par
 Information Management Group\par
 Date\-: 06-\/\-Jun-\/2008\par
 \par
 This utility class can be used to obtain an object property expression in its simplest form. Let P be an object property name and P\-E a property expression, then the simplification is inductively defined as\-: simp(\-P) = P simp(inv(\-P)) = inv(\-P) simp(inv(inv(\-P\-E)) = simp(\-P\-E)
