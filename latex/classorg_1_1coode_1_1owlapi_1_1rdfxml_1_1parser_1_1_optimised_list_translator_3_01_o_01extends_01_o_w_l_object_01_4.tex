\subsection{Detailed Description}
Author\-: Matthew Horridge\par
 The University Of Manchester\par
 Bio-\/\-Health Informatics Group\par
 Date\-: 08-\/\-Jan-\/2007\par
 \par
 

Translates an rdf\-:List into a Java {\ttfamily List}, or Java {\ttfamily Set}. The type of list (i.\-e. the type of objects in the list) are determined by a {\ttfamily List\-Item\-Translator}. The translator consumes all triples which are used in the translation.
