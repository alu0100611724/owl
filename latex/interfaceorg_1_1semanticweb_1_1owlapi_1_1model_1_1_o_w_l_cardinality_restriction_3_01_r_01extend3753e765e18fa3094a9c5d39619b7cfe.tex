Determines if this restriction is qualified. Qualified cardinality restrictions are defined to be cardinality restrictions that have fillers which aren't T\-O\-P (owl\-:Thing or rdfs\-:Literal). An object restriction is unqualified if it has a filler that is owl\-:Thing. A data restriction is unqualified if it has a filler which is the top data type (rdfs\-:Literal). \begin{DoxyReturn}{Returns}
{\ttfamily true} if this restriction is qualified, or {\ttfamily false} if this restriction is unqualified.
