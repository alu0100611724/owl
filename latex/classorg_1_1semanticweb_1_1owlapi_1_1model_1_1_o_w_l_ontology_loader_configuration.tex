\subsection{Detailed Description}
Author\-: Matthew Horridge\par
 The University of Manchester\par
 Bio-\/\-Health Informatics Group\par
 Date\-: 15/12/2010 

A configuration object that specifies options and hints to objects that load O\-W\-L\-Ontologies. Every {\ttfamily \hyperlink{classorg_1_1semanticweb_1_1owlapi_1_1model_1_1_o_w_l_ontology_loader_configuration}{O\-W\-L\-Ontology\-Loader\-Configuration}} is immutable. Changing a setting results in the creation of a new {\ttfamily \hyperlink{classorg_1_1semanticweb_1_1owlapi_1_1model_1_1_o_w_l_ontology_loader_configuration}{O\-W\-L\-Ontology\-Loader\-Configuration}} with that setting. For example, 
\begin{DoxyPre}
\hyperlink{classorg_1_1semanticweb_1_1owlapi_1_1model_1_1_o_w_l_ontology_loader_configuration}{OWLOntologyLoaderConfiguration} config = new \hyperlink{classorg_1_1semanticweb_1_1owlapi_1_1model_1_1_o_w_l_ontology_loader_configuration_a4b6025be25f128f3d35939b4d8eef872}{OWLOntologyLoaderConfiguration()};
config = config.setLoadAnnotationAxioms(false);
\end{DoxyPre}
 creates an {\ttfamily \hyperlink{classorg_1_1semanticweb_1_1owlapi_1_1model_1_1_o_w_l_ontology_loader_configuration}{O\-W\-L\-Ontology\-Loader\-Configuration}} object with the load annotation axioms set to {\ttfamily false}.
