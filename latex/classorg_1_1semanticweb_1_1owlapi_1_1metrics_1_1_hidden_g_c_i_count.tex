\subsection{Detailed Description}
Author\-: Matthew Horridge\par

 Bio-\/\-Health Informatics Group\par
 Date\-: 13-\/\-Aug-\/2007\par
 \par
 

Counts the number of \char`\"{}hidden\char`\"{} G\-C\-Is in an ontology imports closure. A G\-C\-I is regarded to be a \char`\"{}hidden\char`\"{} G\-C\-I if it is essentially introduce via an equivalent class axiom and a subclass axioms where the L\-H\-S of the subclass axiom is nameed. For example, A equivalent\-To p some C, A sub\-Class\-Of B results in a \char`\"{}hidden\char`\"{} G\-C\-I.
