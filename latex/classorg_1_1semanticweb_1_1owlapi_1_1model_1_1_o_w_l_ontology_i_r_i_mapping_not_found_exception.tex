An exception which describes the situation where no ontology document \hyperlink{classorg_1_1semanticweb_1_1owlapi_1_1model_1_1_i_r_i}{I\-R\-I} mapping could be found. This is a runtime exception since clients should really ensure that a mapping exists before attempting to load an ontology.
