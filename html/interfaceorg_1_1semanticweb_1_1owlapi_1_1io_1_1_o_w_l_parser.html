Author: Matthew Horridge
The University Of Manchester

An OWLParser parses an ontology document into an OWL API object representation of an ontology.

OWLOntology  ontology 
) throws OWLParserException, IOException, OWLOntologyChangeException, UnloadableImportException

Parses the ontology that has a concrete representation which is pointed to by the specified document IRI. Implementors of this method should load any imported ontologies with the loadImports method on OWLOntologyManager.

Parameters
documentIRIThe document IRI where the ontology should be loaded from
ontologyThe ontology that the concrete representation should be parsed into.
Returns
An OWLOntologyFormat which describes the concrete representation format which was parsed to obtain the ontology. This will never be null.
The format of the ontology
Exceptions
OWLParserExceptionif there was a problem parsing the ontology. This indicates an error in the syntax with this ontology document that the parser reads.
IOExceptionif there was an IOException during parsing
OWLOntologyChangeExceptionif there was a problem updating the specified ontology from information that was parsed.
UnloadableImportExceptionif loading this ontology prompted the loading of an import and the import could not be loaded.

Parameters

Deprecated:
Each OWLOntology contains a reference to its manager. This method is no longer necessary.
