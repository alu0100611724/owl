Detailed Description

Author: Matthew Horridge
The University Of Manchester
Bio-Health Informatics Group
Date: 02-Jan-2007

Represents the concrete representation format of an ontology. The equality of an ontology format is defined by the equals and hashCode method (not its identity).

OWLOntologyLoaderMetaData org.semanticweb.owlapi.model.OWLOntologyFormat.getOntologyLoaderMetaData ( )

If this format describes an ontology that was loaded from some ontology document (rather than created programmatically) there may be some meta data about the loading process. Subclasses of OWLOntologyFormat will provide accessors etc. to details pertaining to the meta data about loading.

Returns
