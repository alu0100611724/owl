Creates an object duplicator that duplicates objects using the specified data factory and uri replacement map.

Parameters
dataFactoryThe data factory to be used for the duplication.
iriReplacementMapThe map to use for the replacement of URIs. Any uris the appear in the map will be replaced as objects are duplicated. This can be used to "rename" entities.
