OWLProfileReport org.semanticweb.owlapi.profiles.OWLProfile.checkOntology ( OWLOntology  ontology)

Checks an ontology and its import closure to see if it is within this profile.

Parameters
ontologyThe ontology to be checked.
Returns
An OWLProfileReport that describes whether or not the ontology is within this profile.

IRI org.semanticweb.owlapi.profiles.OWLProfile.getIRI ( )

The IRI that uniquely identifies this profile.

If this profile is listed at http://www.w3.org/ns/owl-profile/ this IRI MUST match the IRI given in the list.

Returns
The IRI that identifies this profile.
