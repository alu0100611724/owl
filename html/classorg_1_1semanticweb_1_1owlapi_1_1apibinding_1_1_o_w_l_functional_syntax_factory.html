Detailed Description

Author: Matthew Horridge
The University of Manchester
Information Management Group
Date: 17-Jan-2010

A utility class whose methods may be statically imported so that OWL API objects can be constructed by writing code that looks like the OWL 2 Functional Syntax.

Note that this class is primarily intended for developers who need to write test cases. Normal client code should probably use an org.semanticweb.owlapi.model.OWLDataFactory for creating objects.
