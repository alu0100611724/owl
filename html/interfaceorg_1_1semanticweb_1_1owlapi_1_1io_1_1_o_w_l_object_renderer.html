Detailed Description

Author: Matthew Horridge
The University Of Manchester
Bio-Health Informatics Group
Date: 08-Oct-2007

A general purpose interface, implementations of which can be used to renderer OWL objects (e.g. class expressions, axioms etc.) for presentation in user interfaces, colsole writing etc. etc. Many ontology renderers may also choose to implement this interface.
